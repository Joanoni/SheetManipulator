#### Private helpers

#### `GET /api/tables/{table_name}/rows`
- Validates table existence.
- Applies `WHERE is_deleted = 0` unless `include_deleted=true`.
- Returns `PaginatedRows` with `COUNT(*)` total and `LIMIT/OFFSET` pagination.

#### `POST /api/tables/{table_name}/rows`
- Validates table existence and mandatory columns (HTTP 422 on missing).
- Generates `_row_id = uuid4()`.
- Inserts system columns (`_row_id`, `is_deleted=0`, `_upload_id=NULL`) + user columns in a single `INSERT`.
- Writes one `AuditLog` entry per user column with `operation="INSERT"`.
- Commits in a single transaction; returns inserted row (HTTP 201).

#### `PUT /api/tables/{table_name}/rows/{row_id}`
- Validates table existence; fetches current row (404 if deleted).
- Compares each supplied field to current value; collects only changed fields.
- No-op if nothing changed (returns current row, no DB write).
- Executes `UPDATE` for changed fields only; writes one `AuditLog` entry per changed column with `operation="UPDATE"`.
- Returns updated row (HTTP 200).

#### `DELETE /api/tables/{table_name}/rows/{row_id}`
- Validates table existence; fetches current row.
- Returns `HTTP 409` if already soft-deleted.
- Sets `is_deleted = 1`; writes one `AuditLog` entry: `column_name="is_deleted"`, `old_value="0"`, `new_value="1"`, `operation="DELETE"`.
- Returns HTTP 204 No Content.

#### `GET /api/tables/{table_name}/audit`
- Validates table existence.
- Queries `audit_log` filtered by `table_name` (and optionally `row_id`).
- Orders by `timestamp DESC`, limited by `limit` param (default 100, max 1000).
- Returns `list[AuditLogRead]`.
